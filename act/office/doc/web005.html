配置首页资源

贝美只允许贝美app框架用户配置app首页资源，SDK用户请忽略该功能。
配置首页资源请求说明
http请求方式: POST https://api.bemetoy.com/upload/saveHome post数据包{ token：token data:{ id:"xxx" name:"xxx", type:"type"， url:"xxx", jump_url:"xxxx", content_type:"xx" content_id:"xxx" } }
参数说明
参数 是否必须 说明
token 是 获取到的token
id 否 若不填则表明新建，填了则为修改（int）
name 是 该配置名称(varchar)
type 是 所在位置，滚屏（0）；定制（1）;热播专辑（2）;最新上架（3）;有声绘本（4）；育儿小妙招（5）；早教动画（6）；明星主播（7）；（int）
url 是 图片url(varchar)
jump_url 否 跳转地址(varchar)
content_type 是 资源所在类别：自定义（0），专辑（album）,电台（station）
content_id 否 资源所在类别的id，当配置了jump_url后可不填，其他时间必填
返回说明

正常情况下，贝美会返回下述JSON数据包给（备注：这里的返回值还待定，根据实际需求来定）：

{ token：token data:{ id: "111", name: "xxxx", } }
参数 说明
token 获取到的凭证
data Json形式的数据包
id 首页配置id
name 首页配置名称

错误时贝美会返回错误码等信息，JSON数据包示例如下（该示例为token无效错误）:

{"errcode":10002,"errmsg":"invalid token"}
