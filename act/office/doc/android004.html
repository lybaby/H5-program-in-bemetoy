PEPPER配网

开启配置网络

IOnConfigResultListener listener = new IOnConfigResultListener() { @Override public void onConfigResult(int errCode, String errMsg) { if (errCode == BMErrorCode.Err_None) { //配网成功 } else if (errCode == ErrorCode.Err_Server_Not_Auth) { //未鉴权 } else if (errCode == ErrorCode.Err_Local_Invalid_Args) { //输入参数错误 } else if (errCode == ErrorCode.Err_Net_Response_Timeout) { //配置超时 } else if (errCode == BMConstantValues.BMErrorCode.Err_Net_Anomaly) { //网络异常 } }; BMPepperInfo info = BMPepperInfo.from(ssid, pwd); //禁用声纹 //info.disableAudioWave(); //禁用组播 //info.disableMulticast(); //开启配置 BMIoTClient.getBMPepperConfig().startConfig(info, timeoutMS, listener);
超时时间timeoutMS <= 0时，表示永不超时，此时只有在获得配置结果或者用户主动调用stopConfig函数才会停止配网

停止配网

BMIoTClient.getBMPepperConfig().stopConfig();
