获取贝美云数据

获取资源列表

获取列表接口调用请求说明
http请求方式: POST https://api.bemetoy.com/resource/getList post数据包{ token：token data:{ type:"type"， page:"1" } }
参数说明
参数 是否必须 说明
token 是 获取到的token
type 是 获取相应资源的类别,例如album代表专辑，song代表歌曲，等等
page 否 当前所在页，可不填默认第一页，默认一页20行，如果需要获取第二页的内容请填写该字段
返回说明

正常情况下，贝美会返回下述JSON数据包给（备注：这里的返回值还待定，根据实际需求来定）：

{ token：token data:{ cur_page:"current_page", total :"total_page", type:"type", list: [ { id: "111", name: "xxxx", cnt: "xxxx", } //后续还有同一类型的不同数据，按id排序 ] } }
参数 说明
token 获取到的凭证
data Json形式的数据包
cur_page 当前页
total 该类别的所有条数
type 类别（专辑(album)、电台(station)等等）
list 该类别下的数据列表
id 该类型下的id
name 名称（专辑，电台等等的名称）
cnt 播放数

错误时贝美会返回错误码等信息，JSON数据包示例如下（该示例为token无效错误）:

{"errcode":10002,"errmsg":"invalid token"}

获取单个详情

获取列表接口调用请求说明
http请求方式: POST https://api.bemetoy.com/resource/getDetail post数据包{ token：token data:{ type:"type"， id:"1" } }
参数说明
参数 是否必须 说明
token 是 获取到的token
type 是 获取相应资源的类别,例如album代表专辑，song代表歌曲，等等
id 是 获取相应资源所对应的id
返回说明

正常情况下，贝美会返回下述JSON数据包给（备注：这里的返回值还待定，根据实际需求来定）：

{ token：token data:{ id :"xxxx", name:"xxxx", type:"type", url:"xxxx", song_ids:"xx,xxx,xx", cnt:"xxx" } }
参数 说明
token 获取到的凭证
data Json形式的数据包
type 类别（专辑(album)、电台(station)等等）
song_ids 该对象中对应的歌曲id
id 该类别所在id
name 名称（专辑，电台等等的名称）
cnt 播放数

错误时贝美会返回错误码等信息，JSON数据包示例如下（该示例为token无效错误）:

{"errcode":10002,"errmsg":"invalid token"}
